# MyStore-Webdriverio

This repository contains a webdriverIO-v7 (Node.js/TypeScript) project and libraries using the Cucumber (v7.x) BDD framework to verify the register,signin and purchase product functionality of My store website.

Requirements
1.Node version v12.16.1 or higher

How to run the test
1. Clone the repository to a folder.
2. Open the command line prompt and navigate to the project folder and open it.
3. Run **npm install** to install all the dependencies.
4. After completion then Type **npm run test** to execute the testcases in chrome and edge browser.
5. Have used Allure to generate the report. Type **npm run open-allure** to open the report.
